# www.charliehapersucks.com

## Two and half men Season 4, Episode 16

Un ejercicio de "arqueología web" que trae de vuelta era del Esqueuomorfismo (Skeuomorphism) y los diseños maquetados con <tables>, típicos de finales de los 90 y principios de los 2000. Botones con gradientes exagerados para parecer 3D, fuentes de alto impacto y colores saturados.

![screenshot](https://github.com/lowpez/www.charliehapersucks.com/blob/main/charliehapersucks.com.jpg)
![screenshot B](https://github.com/lowpez/www.charliehapersucks.com/blob/main/charliehapersucks.com-b.jpg)
![screenshot C](https://github.com/lowpez/www.charliehapersucks.com/blob/main/charliehapersucks.com-c.jpg)

Aunque la imagen original probablemente se hizo con tablas HTML antiguas, voy a recrearlo usando las mejores prácticas modernas y Bootstrap 5.3. Mantendremos la estética visual "retro/kitsch" exactamente como se ve, pero bajo el capó será:

1. Responsive: Se adaptará a móviles (las columnas se apilarán).
2. Semántico: Usaremos etiquetas nav, main, h1 en lugar de tablas.
3. Bootstrap Grid System: Para la estructura de columnas.

Dale un vistazo en [Codepen.io](https://codepen.io/lowpez/pen/emzyNGv).
